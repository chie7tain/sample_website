Important HTML tags

This page is designed as a quick reference of some of the common HTML Tags we have covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTMl tags, but they do just many of the most important ones.

Block Elements

> >
Tag Name Purpose
